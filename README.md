# Calculadora

Fiz essa calculadora usando HTML, CSS e JavaScript puro pra praticar lógica e manipulação do DOM.

## 🧠 O que ela faz ?

- Faz contas básicas (+, -, ×, ÷, %)
- Mostra o resultado enquanto você digita
- Guarda um histórico dos cálculos
- Dá pra clicar no histórico e reutilizar a conta
- Interface em modo escuro
- Funciona em diferentes tamanhos de tela

## Tecnologias Usadas

- HTML
- CSS
- JavaScript

## Estrutura do projeto
index.html
style.css
main.js

## Por que fiz isso?

Queria sair do básico e fazer algo mais completo, com interface melhor e algumas funcionalidades a mais (tipo histórico e preview do resultado).
